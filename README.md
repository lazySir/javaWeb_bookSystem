# javaWeb_bookSystem
javaWeb书城后台管理系统

# 运行环境
IntelliJ IDEA 、 mysql 、tomcat

# 语言
JavaWeb 、 servlet 

# 包

c3p0 、echarts ajax

# 运行项目

1.git clone https://github.com/lazySir/javaWeb_bookSystem.git

2.创建bookdb数据库

3.运行sql文件

4.修改c3p0里的user和password

5.配置tomcat 和 各种lib包
